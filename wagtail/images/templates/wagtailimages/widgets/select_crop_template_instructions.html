please pass a template to the SelectCropBlock constructor where you include it... e.g:

my_image_field = SelectCropBlock(required=True, template="your_folder/your_image_spec.html")

The template you specify should define your image specification and the image can be accessed as 'value.image' ... e.g:

{% templatetag openblock %} load wagtailimages_tags {% templatetag closeblock %}
{% templatetag openblock %} image value.image max-600x600 {% templatetag closeblock %}
